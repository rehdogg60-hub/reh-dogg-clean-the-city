# Reh Dogg: Clean the City

A mobile-friendly browser tap game based on Reh Dogg: The Loofah Vigilante.

## How to Play

Tap dirty trouble around the city before the timer runs out.

- Trash = +10
- Criminal mask = +15
- Dirty money = +20
- Corruption badge = +25
- Soap bubble = +30
- Maggie freezes trouble
- Chip clears the screen
- Civilian = -25, avoid tapping

## Features

- Mobile optimized layout
- Sound effects
- Best score saving
- No external libraries
- Works on GitHub Pages
